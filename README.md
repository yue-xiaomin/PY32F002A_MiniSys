#普冉PY32F002A 

PY32F002A 标称是20KB Flash + 3KB SRAM, 但是实际上现在买到的都是 32KB Flash + 4KB SRAM
开发上, 直接使用PY32F030F16的设置就可以了, PY32F030的外设在 PY32F002A 上都有, 已经在 TSSOP20 封装的 PY32F002AF15P 上验证

##PY32F002A运行频率

PY32F002AF15P 支持PLL, 实际直接可用的系统内置时钟为 48MHz, 如果使用外置晶振, 频率也许能更高

##PY32F002A外设

PY32F002AF15P 支持DMA，PY32F002AF15P 其实和 PY32F030 的外设是一样的, 按 PY32F030 的寄存器地址读写都有反应

##PY32F002A/B功耗

PY32F002A: 最低 4.5 uA
PY32F002B: 最低 1.5 uA
